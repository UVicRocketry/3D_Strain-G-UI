# 3D_Strain-G-UI

A 3D PyQT/PyQTGraph GUI for rocket fuselage bending analysis using strain gauges

# How To Run :rocket

1. Run `setup.bat` to install dependencies, setup python local environment, etc..
2. Run `run.bat` to run program
